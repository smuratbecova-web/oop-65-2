Команды только для одно проекта
1) Это команды для инициализации Git в вашем новом проекте, пишется в один раз в начале после создания проекта:
git init

2) Это комада будет вставить вашу ссылку на ваш репозиторий:
git remote add origin https://github.com/arzy-54-2/oop-65-2-prepod.git




Основные команды для работа с Git

1) проверки состояния ваших файлов в проекте.
git status

2) Это команда для добавления в буфер ваших новых файлов и изменений в файлах. после команды add через
пробел ставите точку для того чтобы добавить все файлы:
git add .

3) Это команда для сохранения ваших данных которые находятся в буфере. Внутри кавычек пишите название вашего комита:
git commit -m "Commit name"

4) Это команда для заливки вашего проекта на ваш репозиторий. После команды origin через пробел
пишите название вашей ветки, оно отображается в вашем терминале в скобке (master), (main) или (что-то другое):

git push origin master/main
